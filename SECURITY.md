# Politique de sécurité

## Versions prises en charge

Seules les **dernières versions** des applications Upluce (web et desktop) reçoivent des correctifs de sécurité.

## Signaler une vulnérabilité

⚠️ **N'ouvrez pas d'issue publique** pour une faille de sécurité.

Contactez-nous en privé :

- **Email** : admin@upluce.com
- ou via **GitHub Security Advisories** (bouton « Report a vulnerability » de l'onglet *Security* du dépôt concerné).

Merci d'inclure : description et impact, étapes de reproduction (PoC si possible), versions/composants affectés.

## Engagement

- Accusé de réception sous **72 h**.
- Suivi jusqu'au correctif, **divulgation coordonnée** (merci de nous laisser un délai raisonnable).
